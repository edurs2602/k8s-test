# SKILLS.md

## Project Skills & Competencies

### Backend Technologies

#### FastAPI
- **Skill Level**: Intermediate to Advanced
- **Description**: Modern, fast web framework for building APIs with Python 3.11+
- **Key Features Used**:
  - Async/await support for high performance
  - Automatic OpenAPI documentation
  - Pydantic integration for data validation
  - Dependency injection system
  - CORS middleware configuration

#### SQLAlchemy 2.0 (Async)
- **Skill Level**: Intermediate
- **Description**: SQL toolkit and ORM with async support
- **Key Features Used**:
  - Async engine sessions
  - Declarative base models
  - Mapped columns with type hints
  - Async query patterns

#### Pydantic v2
- **Skill Level**: Intermediate to Advanced
- **Description**: Data validation using Python type annotations
- **Key Features Used**:
  - BaseModel for request/response schemas
  - Settings management with pydantic-settings
  - model_validate for ORM model conversion
  - ConfigDict for model configuration

#### UV Package Manager
- **Skill Level**: Beginner to Intermediate
- **Description**: Fast Python package installer and resolver
- **Key Features Used**:
  - Dependency management via pyproject.toml
  - Fast installation speeds
  - Virtual environment management

### Frontend Technologies

#### Vue.js 3 (Composition API)
- **Skill Level**: Intermediate
- **Description**: Progressive JavaScript framework
- **Key Features Used**:
  - Composition API with `<script setup>`
  - Reactive state management
  - Component-based architecture
  - Template syntax with v-for, v-if

#### TypeScript
- **Skill Level**: Intermediate
- **Description**: Typed superset of JavaScript
- **Key Features Used**:
  - Interface definitions
  - Type annotations
  - Generic types
  - Type guards

#### Pinia
- **Skill Level**: Beginner to Intermediate
- **Description**: Vue state management library
- **Key Features Used**:
  - Composition store setup
  - Reactive refs
  - Actions for async operations

#### Vite
- **Skill Level**: Beginner to Intermediate
- **Description**: Next-generation frontend build tool
- **Key Features Used**:
  - Fast development server
  - Hot module replacement
  - Production builds
  - Proxy configuration for API

### Infrastructure

#### Docker
- **Skill Level**: Intermediate
- **Description**: Container platform for packaging applications
- **Key Features Used**:
  - Multi-stage builds
  - Image optimization
  - Docker Compose for local development
  - Environment configuration

#### Kubernetes (AKS)
- **Skill Level**: Beginner to Intermediate

- **Key Features Used**:

  - Service definitions
  - Ingress configuration

- **Key Features Used**:
